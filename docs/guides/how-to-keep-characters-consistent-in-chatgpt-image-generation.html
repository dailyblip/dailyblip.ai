How to Keep Characters Consistent in ChatGPT Image Generation

Keep the same face, outfit, and style across a whole set of images.

Last reviewed July 20, 2026
Image GenerationGetting Started

ChatGPT doesn't remember a character between separate conversations. The fix is giving it the same reference every time, not hoping it recalls.

Give it a reference to work from

Upload one clear reference image of the character and keep reusing that exact same upload for every new image.

Describe the fixed details every time too: hair color, outfit, build. Don't assume the model remembers them from your last message.

Ask it to edit the existing image rather than generate a new one when the change is small, like a different pose or background.

Lock down the details that tend to drift

Clothing color and hairstyle are the two things most likely to shift between generations. Restate both explicitly each time.

Say "preserve facial features exactly as in the uploaded photo" directly in your prompt. It is a real instruction, not filler.

If a generation drifts, don't keep pushing forward. Go back to the last version that looked right and edit from there instead.

Consistency isn't a setting you turn on. It's just discipline about which image you build the next one from.

PROMPTS TO TRY
Reference-based edit
Using the uploaded photo, change the background to a city street at night, preserve facial features, hairstyle, and outfit exactly as shown
Restating fixed details
The same woman with short brown hair and a red jacket from the previous image, now standing in a park, same face and proportions
Small pose change only
Same character and outfit as the uploaded reference, now facing slightly to the left, everything else unchanged
KEY TAKEAWAYS
  • Reuse the same reference image every time, not just a text description.
  • Explicitly restate hairstyle and clothing color in every new prompt.
  • Edit the last good version instead of generating fresh when a small detail drifts.
